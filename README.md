## 📝 Descripción del Proyecto
MediVecino es una solución tecnológica diseñada para mitigar el olvido en la toma de medicamentos. Mediante el uso de alertas inteligentes, escaneo de recetas (OCR) y un sistema de monitoreo familiar, buscamos apoyar a los pacientes (especialmente adultos mayores) a cumplir con sus tratamientos médicos de manera segura y eficiente.

## 🛠️ Tecnologías Propuestas
* **Frontend:** Flutter / React Native (o la que hayan elegido)
* **Backend:** Node.js / Python
* **Base de Datos:** Firebase / MongoDB
* **Servicios:** Cloud Computing / IA para lectura de recetas
